Strawss Berry

About

2D platform game where the player has to solve puzzles by posessing enemies. The engine was built from the ground up using C++ and OpenGL.

Team

Programmers: Pablo Marcos, Oscar Calletano, Ignacio Mendezona, Tomas Lacasa, Gaizka Calvo
Artists: Peio Ureta, Inaki Ugarte, Eder Bilbao, Jon Cascales, Elena Marta Garcia
Musician: Jorge Damien Fernandez

Contribution

Engine: I built the hole engine by myself, excluding the event system. This includes: graphics, serialization, resource management, physics,2D animation system, editor...
Gameplay: Final boss, menus, player's basic movement, some of the enemies.
Late Design: I fixed all the bugs, gave the game an ending and prepared the game to meet our university's gallery submision guidelines.
