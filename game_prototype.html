Game Prototype

About

A game I have been working on lately on my own. I use Unreal Engine favoring the use of C++ instead of Blueprints whenever possible. It features the following elements:

AI

The AI uses Unreal's Perception Stimulus Source Component to sense the environment. It gets triggered by either seeing, hearing, touching or being damaged by the player.

  • Sight

  • Sound

  • Touch

  • Damage

Procedural Terrain Generation

When the player enters an elevator and the doors close, the remaining level is destroyed and a new level is generated. This is done using pre built level tiles stored in non persistent levels. Items which where droped in the level get destroyed unless they are in the elevator. Afterwards, enemies are spawned through the level. When everything is set up the elevator doors open.

Weapons and Items

As of now, there are 4 types of weapons.

  • Pistol

  • Rifle

  • Other Rifle

  • Shotgun

It is allowed to smoke in this game. It doesn't do anything though. Smoking with no repercussions, yay!
