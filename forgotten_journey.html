Forgotten Journey

About

Puzzle oriented platformer in which you solve puzzles and climb walls. Made with Unreal Engine 4.

Team

Programmers: Javier Tafur, Pablo Marcos, Blai Navarro, Monica Marquez, Ignacio Mendezona, Leo Mogenet
Artists: Peio Ureta, Desiree Maroto, Ane Unzalu, Eneko Agudo, Eder Bilbao
Musician: Jorge Damien Fernandez

Contribution

I implemented all the effects in the game using the Niagara VFX system and added most of the sounds. I also implemented the player's equilibrium state and contributed to its basic movement (not the climbing).
